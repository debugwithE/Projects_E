# Number Guessing Game

## 1. Overview:
There is a secret number which the user has to guess. Based on the number entered by the user, it gives feedback on whether the number guessed is correct, high or low. If the number guesses is correct, it displays a congratulatory message along with the number of attempts in which the correct number was guessed, and the game is terminated.

-> Strenthening Problem Sloving Fundamentals
